## Описание
Консольное Java-приложение для анализа использования дискового пространства.  
Приложение автоматически определяет операционную систему, выполняет соответствующую системную команду (`df` для Linux/macOS или `wmic` для Windows), парсит вывод и формирует JSON-отчет.

Отчет содержит:
1. Список дисков, заполненных выше заданного порога.
2. Общую сводку по всем дискам (Всего GB, Использовано GB).

## Технологии
*   Java 17
*   Gradle (сборщик)
*   Gson (работа с JSON)
*   JUnit 5 (тестирование)
